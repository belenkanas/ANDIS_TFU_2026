# Truck & Roll

Backend REST para el sistema de gestión de food trucks, desarrollado en Python con FastAPI.

## Instalación

```bash
py -m pip install -r requirements.txt
```

## Ejecutar la API

```bash
py -m uvicorn main:app --reload
```

La API queda disponible en `http://localhost:8000`
